Basic game developped in Programmation 2 course
Run the executable to play :
Cpp_Diamonds\Diamonds_game\

MOVES
===
Move with arrows from
arrow keyboard
or
numeric keypad (including diagonals)
===

TILE COLORS
===
Blue --> move is possible
Pink --> one last move possible on the tile
Black --> "wall": no access
Green $$$ --> money
===
